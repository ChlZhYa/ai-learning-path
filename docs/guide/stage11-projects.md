## 11.1 推荐实战项目

**项目一：个人知识库 Agent（基础）**
- 技术栈：OpenAI Agents SDK + ChromaDB + FastAPI
- 功能：上传文档 → 自动索引 → 自然语言问答 → 引用来源
- 价值：掌握 RAG 全流程

**项目二：代码审查 Agent（中级）**
- 技术栈：LangGraph + GitHub API + MCP
- 功能：监听 PR → 分析代码变更 → 生成审查意见 → 自动评论
- 价值：掌握工具调用、工作流编排

**项目三：多 Agent 研究助手（高级）**
- 技术栈：CrewAI + Web 搜索 + PDF 解析
- 功能：用户提问 → 多 Agent 协作研究 → 生成报告
- 价值：掌握多 Agent 协作

**项目四：智能客服系统（生产级，大厂面试重点）**
- 技术栈：FastAPI + Redis + PostgreSQL + Milvus + Docker
- 功能：多轮对话 → 意图识别 → 工单系统 → 人工转接
- 价值：掌握完整生产部署
