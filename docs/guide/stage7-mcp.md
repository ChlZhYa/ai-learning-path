## 7.1 MCP（Model Context Protocol）

> [!] 阿里 JD 明确要求"理解 MCP、A2A、ANP 等智能体通信协议"

MCP 是 Anthropic 推出的开放协议，定义 AI 模型与外部数据源/工具之间的标准化接口，类似 AI 领域的"USB 接口"。
