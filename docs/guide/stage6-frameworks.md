## 6.1 框架选型指南


| 框架 | 核心优势 | 学习难度 | 大厂使用 |
|------|---------|---------|---------|
| **LangChain/LangGraph** | 生态最全，灵活 | ★★★ | [OK] 最广泛 |
| **OpenAI Agents SDK** | 官方支持，简洁 | ★ | [OK] 新项目 |
| **CrewAI** | 多 Agent 角色化 | ★★ | 试点中 |
| **Dify** | 低代码，可视化 | ★ | [OK] 国内流行 |
| **Coze（扣子）** | 字节出品，低代码 | ★ | [OK] 字节生态 |

**大厂偏好：** 字节偏自研框架 + LangGraph，阿里偏 LangChain + 自研，腾讯偏 LangGraph，百度偏 Dify/自研平台。

## 6.4 国产平台实战 

| 平台 | 厂商 | 特点 |
|------|------|------|
| **Coze（扣子）** | 字节跳动 | 低代码 Agent，插件生态丰富 |
| **Dify** | 开源 | 可视化工作流，支持自部署 |
| **百度 AgentBuilder** | 百度 | 零代码智能体构建 |
| **阿里云百炼** | 阿里 | 全链路 LLM 开发平台 |
| **华为云 ModelArts Studio** | 华为 | 一站式大模型开发平台 |
